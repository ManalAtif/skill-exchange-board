My Listings

src/pages/MyListings.jsx

Lets logged-in users manage all listings they've posted — view, edit, or delete each one.

  • Skeleton loader while fetching

  • Toast notification on successful delete
  • "New listing" CTA button in the page header

New Component

Toast Notification System

src/components/Toast.jsx

Global toast notifications shown in the bottom-right corner for success and error feedback.

  • Context-based — any component can call addToast(message, type)
  • Success (green) and error (red) variants
  • Auto-dismisses after 3.5 seconds
  • Manual dismiss button
  • Framer Motion enter/exit animations
  • Used on: Profile save, listing delete, listing report, edit listing, block user

Design System

Visual Design Overhaul

The CSS design tokens and component styles were significantly upgraded to give the whole app a more professional, polished feel.

Styles

Design Tokens

src/styles/tokens.css

The global CSS variables were upgraded with a richer palette, better typography, and a 4-level shadow system.

  • Google Fonts loaded directly: Fraunces (display), IBM Plex Sans (body), IBM Plex Mono
  • Expanded colour palette: teal, rust, gold — each with light/dark/bg tints
  • 4-level shadow scale: xs, card, card-hover, pop
  • Larger border radii (up to 26px) for a premium feel
  • Subtle crosshatch SVG pattern on the corkboard hero texture
  • Added section-label utility class used across all pages
Styles

Component Styles

src/styles/components.css

All component classes were refined for a more professional look and feel.

  • Buttons: lift on hover with coloured drop shadows
  • Inputs: focus glow ring (not just border change)
  • Select element: custom dropdown arrow via background SVG
  • Stamp badges: now have tinted background fills
  • Pin cards: smoother rotation (3n pattern), scale on hover
  • Confirm dialog: blur backdrop
  • Skeleton loader shimmer animation
  • Stats grid responsive breakpoints (4 col → 2 col → 1 col)
  • ListingCard "View details →" footer with animated arrow gap

Summary

What Was Delivered

This contribution transformed an existing MVP frontend into a complete, professional-grade application with the following improvements:

  • 4 new fully functional pages (NotFound, MyListings, UserProfile, EditListing)
  • 4 new reusable components (Footer, SkeletonCard, Toast, ConfirmDialog)
  • Complete visual redesign of Home, Login, and Signup pages
  • Enhanced UX on Browse, Chat, Listing Detail, Matches, Profile, and Navbar
  • A professional design system with Google Fonts, richer colours, and polished shadows
  • Full mobile responsiveness across all new and modified pages
  • All pages work with mock data — no backend required for demonstration
